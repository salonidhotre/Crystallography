The Seven Crystal Systems

Introduction

The smallest possible unit that repeats itself in a crystal lattice is the “unit cell”.
Let's say you are given three sets of planes and three vectors as ‘a’, ‘b’ and ‘c’.  How many types of unit cell can you form?

A crystallographer named Auguste Bravais suggested that 14 structures can be formed; these 14 structures are known as Bravais Lattices.

Following is an attempt to illustrate these 14 Bravais Lattices of the 7 Crystal systems.
You can use the cursor to drag and enjoy the immersive view of these lattices.

Note: The objects may take time to load.

The Seven Crystal Systems

1. Cubic
2. Tetragonal
3. Orthorhombic
4. Rhombohedral
5. Monoclinic
6. Triclinic
7. Hexagonal




1. Cubic

a = b = c     α = β = γ = 90°

Primitive

Examples: Polonium, Oxygen, Fluorine

Body Centered Cubic BCC

Examples: Lithium, Sodium, Potassium, Iron, Molybdenum, Barium, Tungsten, Manganese

Face Centered Cubic FCC

Examples: Neon, Aluminum, Silicon, Argon, Calcium, Nickel, Copper, Krypton, Silver, Platinum, Gold, Lead

2.Tetragonal

a = b ≠ c     α = β = γ = 90°

Examples: Indium, Tin

Primitive

Body Centered Tetragonal

3. Orthorhombic

a ≠ b ≠ c     α = β = γ = 90°

Examples: Chlorine, Galium, Bromine, Iodine, Uranium, Sulfur

Primitive

Body Centered Orthorhombic

Face Centered Orthorhombic

Side Centered Orthorhombic

4. Rhombohedral also called Trigonal

a = b = c     α = β = γ ≠ 90°

Primitive

Examples: Boron, Antimony, Mercury, Arsenic

5. Monoclinic

a ≠ b ≠ c     α = β = 90°    γ ≠ 90°

Examples: Plutonium (synthetically prepared)

Primitive

Side Centered Monoclinic

6. Triclinic

a ≠ b ≠ c     α ≠ β ≠ γ ≠ 90°

Primitive

Example: Fool's Gold (FeS2)

7. Hexagonal

a = b ≠ c     α = β = 90°    γ = 120°

Primitive

Examples: Carbon, Nitrogen
